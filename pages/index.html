Bring NextJS Page Router to HTML

Use the .JS file from the repo, to enable nextjs style page router in your HTML based websites.

It uses the same `pages/` directory, in which you can create multiple .html pages and also use folders to nest these files for better organization 

/pages
    ├── index.html
    ├── dashboard
    │   └── index.html
    │   └── home.html
    └── 404.html
